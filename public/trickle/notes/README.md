# Almost Real - Fine Art Photography Gallery

## Project Overview

This is a clone of the Almost Real photography gallery website (https://almostreal.me/), featuring fine art photography and visual arts by a collective of global artists and creatives.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Navigation**: Sticky header with dropdown menus for Collections and Artists
- **Hero Section**: Full-screen hero with background image and call-to-action buttons
- **Featured Products**: Grid layout showcasing photography prints with hover effects
- **Mobile Menu**: Slide-out mobile navigation menu
- **Search Functionality**: Expandable search bar in header
- **Footer**: Comprehensive footer with links and social media icons

## Design Style

- **Modern Minimalist**: Clean, professional design with generous white space
- **Typography**: Inter font family for modern readability
- **Colors**: Black and white primary palette with subtle gray accents
- **Images**: High-quality photography from Unsplash
- **Interactions**: Smooth hover effects and transitions

## Navigation Structure

- Collections: All Prints, Automobile, Black & White, Books, Film, Nature, People, Travel, Water, Apparel
- Artists: Multiple featured photographers including Anna Landstedt, Cal Foster, Caley Vanular, and others
- Additional Pages: About, Meet The Artist, Snaaps 35mm, Contact

## Technical Implementation

- **React 18**: Component-based architecture
- **TailwindCSS**: Utility-first styling
- **Lucide Icons**: Modern icon library
- **Responsive Grid**: CSS Grid and Flexbox layouts
- **Error Boundaries**: Robust error handling
